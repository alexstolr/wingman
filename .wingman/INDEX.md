# Wingman — Agent Entry Point

This folder is the AI capability store for everything — work, personal life, automations, and any task the agent handles. All rules, skills, tools, hooks, agents, personas, prompts, souls, conventions, and workflows live here. **Always check these capabilities first and apply them whenever relevant before proceeding with any task. Conventions are not optional — they must be strictly followed and applied at all times.**
